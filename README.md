# Northwind Traders Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard built using the Northwind Traders dataset to analyze sales, customers, products, suppliers, and business performance.

---

## Tools Used
- Power BI
- SQL
- Excel
- PowerPoint

---

## Dashboard Pages
- Customer Analytics
- Product Analytics
- Sales Analytics
- Country Analytics

---

## Key Insights
- Customer distribution across countries
- Sales trend analysis over time
- Product and supplier performance tracking
- Regional business insights
- Interactive business intelligence reporting

---

## Files Included
- Power BI Dashboard (.pbix)
- PowerPoint Presentation
- MECE Breakdown Document
- Excel Dataset
- Power BI Answers
